# Con Ingenio - Agencia de Software y Consultoría

Bienvenido al repositorio oficial de **Con Ingenio**, una Landing Page desarrollada como proyecto académico para demostrar el dominio avanzado de desarrollo Frontend utilizando tecnologías nativas.

## Descripción del Proyecto
Este proyecto consiste en la maquetación y programación de una interfaz web moderna, responsiva y accesible. Se optó por construir la plataforma utilizando **Vanilla CSS y JavaScript puro**, prescindiendo de frameworks externos, para garantizar un control absoluto sobre el diseño, la semántica (HTML5) y la experiencia del usuario (UX).

## Características Principales
- **Diseño Responsivo (Desktop-First adaptado):** Interfaz fluida optimizada para múltiples dispositivos (Desktop, iPad, Smartphones) utilizando CSS Grid y Flexbox.
- **Modo Oscuro (Dark/Light Theme):** Implementación de cambio de tema interactivo con persistencia de datos mediante `localStorage`.
- **Menú Interactivo:** Navegación optimizada con menú hamburguesa dinámico para dispositivos móviles.
- **Validación de Formularios (Client-Side):** Seguridad aplicada en el lado del cliente para prevenir envíos vacíos, validar formatos de correo mediante Regex y exigir longitud mínima de caracteres.
- **UX/UI Refinada:** Navegación por anclas con scroll suave y legibilidad optimizada a 16px bajo estándares WCAG.

## Tecnologías Utilizadas
- **HTML5** (Arquitectura y estructura semántica).
- **CSS3** (Sistema de diseño por variables, Media Queries, transiciones).
- **JavaScript (ES6)** (Manipulación del DOM, Validaciones, Eventos).

## Cómo ejecutar el proyecto localmente
Para revisar el código fuente y visualizar el proyecto en tu máquina:
1. Clona este repositorio usando tu terminal:
   `git clone https://github.com/CrisBustosE/EVA1_Con_Ingenio.git`
2. Abre la carpeta del proyecto.
3. Ejecuta el archivo `index.html` en tu navegador web de preferencia. No requiere instalación de dependencias (Node.js/NPM) ni servidores locales.
